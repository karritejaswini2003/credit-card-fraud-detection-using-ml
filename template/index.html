How It Works — Pipeline
01 · INPUT
Transaction sent
Features + email + phone sent as JSON POST to Flask API
02 · SCALE
StandardScaler
Amount and Time are normalized using the saved scaler.pkl
03 · PREDICT
XGBoost runs
Model returns fraud probability 0–1 in under 10ms
04 · ALERT
Alerts fired
Email via Gmail + SMS via Twilio in background threads
05 · RESPONSE
Result shown
UI updates with fraud verdict, probability, and risk level
